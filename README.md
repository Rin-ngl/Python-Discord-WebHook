**Discord Webhook Sender**

Send messages to Discord channels using webhooks.

**Requirements:**
- Python 3
- Install: `pip install customtkinter discord-webhook pillow requests`

**Creating a Discord Webhook:**
1. Right-click your Discord channel → **Edit Channel**
2. Go to **Integrations** → **Webhooks**
3. Click **New Webhook**
4. Customize name and avatar (optional)
5. Click **Copy Webhook URL**
6. Keep the URL secure - anyone with it can send messages to your channel

**Usage:**
1. Run the script
2. Paste your webhook URL in the top field
3. Choose message type (Simple or Embed)
4. Enter your message
5. Click **Send Message**

**Features:**
- Save webhook URLs for quick access
- Send plain text or rich embed messages
- Customize bot username and avatar
- Attach files
- Add embed fields, images, and more
